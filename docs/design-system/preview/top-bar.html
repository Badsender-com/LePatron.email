Harmonisation de la Barre Supérieure

Unifier l'apparence et la structure du header sur tous les contextes de l'application

Objectif : Harmoniser les 3 variantes de barre supérieure (Admin, Liste emails, Editor) vers une structure unifiée de 48px avec des slots contextuels et les icônes Material Design.

Etat actuel : 3 variantes incohérentes

Problème Actuel Impact
Hauteurs incohérentes 40px / 48px / 64px Rupture visuelle entre les contextes
Bibliothèques d'icônes différentes Material Design (Vue) vs Font Awesome (Editor) Incohérence visuelle, double charge

Design cible : Header unifié avec slots contextuels

Hauteur unique : 48px (compromis entre 40px editor et 64px Vuetify)
Structure : 3 slots (Gauche, Centre, Droite) avec contenu contextuel

Avantages de l'harmonisation

straighten Cohérence

Même hauteur (48px) sur tous les contextes, transition fluide entre les pages.

extension Réutilisabilité

Structure de slots réutilisable, facile à adapter pour de nouveaux contextes.

speed Performance

Une seule bibliothèque d'icônes (MDI) au lieu de deux.

palette White-Label ready

Tokens CSS permettent de changer les couleurs via la config groupe.

code Maintenabilité

Pattern unique à maintenir, moins de code dupliqué.

psychology UX

Utilisateur toujours situé grâce à une structure prévisible.
