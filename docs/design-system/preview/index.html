Principes directeurs

1. Tokens personnalisables, composants fixes — Le white-label ne touche que les couleurs, logo et polices. La structure des composants reste identique pour tous.

2. Harmonisation progressive — Pas de refonte massive. Chaque nouvelle feature ou correction applique le design system cible.

3. Dual-stack respecté — Vue App (Vuetify) et Editor (Knockout/jQuery) partagent les tokens mais gardent leurs composants.
