Composants Editor

Composants de l'éditeur d'email (Knockout.js, jQuery UI, LESS) jQuery UI Knockout Font Awesome 4.7

Stack legacy : L'éditeur utilise une stack différente de l'app Vue. Ces composants utilisent les mêmes tokens de couleur (via CSS variables) mais ont leur propre style défini dans des fichiers LESS.
